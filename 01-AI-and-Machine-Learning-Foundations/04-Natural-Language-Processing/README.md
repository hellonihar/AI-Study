# Natural Language Processing (NLP)

Techniques for processing and understanding human language with machines.

## Key Topics
- Text preprocessing (tokenization, stemming, lemmatization)
- Bag-of-words, TF-IDF, n-grams
- Word embeddings (Word2Vec, GloVe, FastText)
- Sequence labeling (POS tagging, NER, chunking)
- Language modeling (n-gram, neural LM)
- Machine translation and sequence-to-sequence models
- Text classification and sentiment analysis
- Dependency parsing and constituency parsing
- Topic modeling (LSA, LDA)
- Attention mechanisms and self-attention
- Contextual embeddings (ELMo, BERT, RoBERTa)
- Modern NLP evaluation benchmarks
