# Deep Learning Concepts

Neural network architectures and training techniques that power modern AI.

## Key Topics
- Perceptron and multi-layer perceptrons (MLP)
- Backpropagation and gradient descent variants (SGD, Adam, RMSprop)
- Activation functions (ReLU, sigmoid, tanh, GELU, SwiGLU)
- Initialization strategies (Xavier, He)
- Batch normalization, layer normalization, dropout
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs, LSTMs, GRUs)
- Sequence-to-sequence models and attention
- Autoencoders and variational autoencoders (VAEs)
- Generative Adversarial Networks (GANs)
- Diffusion models
- Optimization techniques (warmup, cosine decay, gradient clipping)
