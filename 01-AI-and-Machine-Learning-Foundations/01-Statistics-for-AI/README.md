# Statistics for AI

Core statistical concepts used throughout machine learning and AI.

## Key Topics
- Descriptive statistics (mean, median, variance, standard deviation)
- Probability theory (conditional probability, Bayes' theorem)
- Probability distributions (normal, binomial, Poisson, exponential)
- Hypothesis testing and confidence intervals
- Bayesian inference and priors
- Maximum likelihood estimation (MLE) and maximum a posteriori (MAP)
- Correlation and covariance
- Dimensionality reduction (PCA, SVD)
- Information theory (entropy, KL divergence, mutual information)

## Applications in AI
- Loss functions derive from MLE principles
- Bayesian methods for uncertainty estimation
- Information theory guides feature selection and model evaluation
- Hypothesis testing for A/B testing in AI systems
