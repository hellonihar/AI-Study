# Operations

Day-2 operations for running AI systems reliably at scale.

## Topics
- Deployment pipelines: staging, canary, blue-green, rolling updates
- Monitoring: quality, cost, latency, drift dashboards and alerting
- Incident response: runbooks, on-call rotations, escalation paths, post-mortems
- Cost management: budget controls, usage tracking, anomaly detection, chargeback
- Capacity management: GPU utilization, queue depth, auto-scaling policies
- Model lifecycle: promotion gates, A/B testing, rollback automation, retirement
- SLOs/SLIs: defining and measuring uptime, latency, throughput, accuracy
