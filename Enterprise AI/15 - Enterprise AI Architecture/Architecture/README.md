# Architecture

Reference architectures and design patterns for enterprise AI systems.

## Topics
- Multi-region, multi-AZ deployment topologies
- Capacity planning: GPU sizing, throughput modeling, cost estimation
- Inference infrastructure: vLLM, TGI, Triton, ONNX Runtime
- Model serving patterns: replica sets, tensor parallelism, pipeline parallelism
- Caching strategies: semantic cache, prefix cache, response cache
- Networking: VPC design, private endpoints, bandwidth planning
- Disaster recovery: cross-region failover, data replication, RPO/RTO
- Hybrid cloud: on-premise GPU clusters + cloud burst
