# MCP (Model Context Protocol)

Open protocol for standardizing how LLMs discover and interact with tools, data sources, and resources.

## Key Topics
- MCP specification: resources, tools, prompts, and transports
- MCP server implementation: exposing enterprise data and services
- MCP client integration: connecting agents to standardized tool registries
- Security: authentication, authorization, audit logging per tool call
- Tool discovery: dynamic registration, versioning, deprecation
- Ecosystem: reference servers, SDKs, community integrations
- Vendor-neutral tooling: plug-and-play across model providers
