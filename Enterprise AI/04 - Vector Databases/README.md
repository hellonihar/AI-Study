# Vector Databases

Production-grade vector storage, indexing, and Approximate Nearest Neighbor (ANN) search.

## Key Topics
- ANN index types: HNSW, IVF, PQ, DiskANN, IVFPQ
- Performance benchmarks: latency, recall, QPS, memory usage
- Managed services: Pinecone, Weaviate Cloud, Zilliz
- Self-hosted: Milvus, Qdrant, pgvector, Elasticsearch
- Hybrid search (dense + sparse + metadata filtering)
- Index mutation strategies: streaming inserts, deletes, compaction
- Multi-tenancy, RBAC, and data isolation patterns
