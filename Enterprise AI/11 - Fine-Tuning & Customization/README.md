# Fine-Tuning & Customization

Adapting pre-trained models to domain-specific tasks through supervised fine-tuning and alignment.

## Key Topics
- Data preparation: curation, deduplication, format standardization, quality filtering
- Parameter-efficient fine-tuning: LoRA, QLoRA, Adapters, Prefix Tuning
- Full fine-tuning: when it's justified, infrastructure requirements, checkpointing
- Alignment: RLHF, DPO, ORPO, KTO — when and why
- Evaluation: held-out eval sets, task-specific metrics, catastrophic forgetting checks
- Deployment: vLLM, TGI, ONNX runtime, quantized inference (FP8, INT4)
- Iteration: continuous fine-tuning, data flywheel, regression testing
