# Enterprise Data Architecture

Designing data pipelines and storage systems that feed AI workloads at scale.

## Key Topics
- Structured data: SQL sources, CDC pipelines, data warehouses (Snowflake, BigQuery, Redshift)
- Unstructured data: PDFs, Confluence, email, SharePoint, web crawls
- Data freshness SLAs: batch vs. streaming ingestion, delta syncs
- Schema evolution, data versioning, and lineage tracking
- Data quality: deduplication, validation, anomaly detection
- Privacy: PII identification, data masking, retention policies
- Lakehouse architecture (Databricks, Delta Lake, Iceberg)
