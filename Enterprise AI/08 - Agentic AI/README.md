# Agentic AI

Autonomous LLM agents that perceive, reason, act, and iterate to accomplish tasks.

## Key Topics
- Agent loop: observe → plan → act → reflect
- Tool use: API calls, code execution, DB queries, browser automation
- Memory: short-term (conversation window), long-term (vector store), episodic
- Planning: ReAct, Plan-and-Solve, Tree-of-Thoughts, Reflexion
- Self-correction: hallucination detection, retry logic, human-in-the-loop
- Evaluation: task completion rate, cost per task, steps per task
- Safety: sandboxing, tool permissions, rate limits, budget controls
