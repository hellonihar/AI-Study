# Resources — Agentic AI

## Foundational Papers

| Paper | Year | Contribution |
|-------|------|-------------|
| [ReAct: Synergizing Reasoning and Acting in Language Models](https://arxiv.org/abs/2210.03629) | 2022 | ReAct framework — interleaved reasoning + actions |
| [Reflexion: Language Agents with Verbal Reinforcement Learning](https://arxiv.org/abs/2303.11366) | 2023 | Self-reflection for agent improvement |
| [Tree of Thoughts: Deliberate Problem Solving with LLMs](https://arxiv.org/abs/2305.10601) | 2023 | Tree search over reasoning paths |
| [Plan-and-Solve Prompting](https://arxiv.org/abs/2305.04091) | 2023 | Decomposition + planning for complex tasks |
| [LLM as a Tool Maker](https://arxiv.org/abs/2305.17118) | 2023 | Agents that create their own tools |
| [Voyager: An Open-Ended Embodied Agent with LLMs](https://arxiv.org/abs/2305.16291) | 2023 | Lifelong learning agent for Minecraft |
| [AgentBench](https://arxiv.org/abs/2308.03688) | 2023 | Benchmark for evaluating LLM agents |
| [AutoGen: Enabling Next-Gen LLM Applications](https://arxiv.org/abs/2308.08155) | 2023 | Multi-agent conversation framework (Microsoft) |
| [TaskWeaver: A Code-First Agent Framework](https://arxiv.org/abs/2311.17541) | 2023 | Code-centric agent design (Microsoft) |
| [OpenDevin: Code Less, Make More](https://arxiv.org/abs/2407.16741) | 2024 | Open-source autonomous coding agent |
