# Resources

## Foundational Papers

| Paper | Year | Key Contribution |
|---|---|---|
| [Attention Is All You Need](https://arxiv.org/abs/1706.03762) | 2017 | Transformer architecture |
| [BERT: Pre-training of Deep Bidirectional Transformers](https://arxiv.org/abs/1810.04805) | 2018 | Encoder-only pretraining |
| [Language Models are Few-Shot Learners (GPT-3)](https://arxiv.org/abs/2005.14165) | 2020 | Scaling, in-context learning |
| [Scaling Laws for Neural Language Models](https://arxiv.org/abs/2001.08361) | 2020 | Kaplan scaling laws |
| [Training Compute-Optimal Large Language Models (Chinchilla)](https://arxiv.org/abs/2203.15556) | 2022 | Optimal token:param ratio |
| [LLaMA: Open and Efficient Foundation Language Models](https://arxiv.org/abs/2302.13971) | 2023 | Open-source scaling recipes |
| [Mistral 7B](https://arxiv.org/abs/2310.06825) | 2023 | Efficient small model design |

## Tools & Libraries

| Tool | Description |
|---|---|
| [Ollama](https://ollama.com) | Run local models with one command |
| [vLLM](https://github.com/vllm-project/vllm) | High-throughput inference with PagedAttention |
| [llama.cpp](https://github.com/ggerganov/llama.cpp) | CPU/edge inference with GGUF |
| [Hugging Face Transformers](https://github.com/huggingface/transformers) | Universal model interface |
| [GPTQ](https://github.com/AutoGPTQ/AutoGPTQ) | Post-training quantization |
| [AWQ](https://github.com/mit-han-lab/llm-awq) | Activation-aware weight quantization |
| [llama-cpp-python](https://github.com/abetlen/llama-cpp-python) | Python bindings for llama.cpp |
