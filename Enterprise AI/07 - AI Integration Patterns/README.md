# AI Integration Patterns

Patterns for integrating LLMs into existing enterprise systems reliably and cost-effectively.

## Key Topics
- Decision framework: RAG vs. fine-tuning vs. prompt engineering
- Tool-calling agents: function definitions, parameter binding, error recovery
- Fallback chains: tiered model routing (cheap → expensive)
- Circuit breakers: graceful degradation when models or APIs fail
- Streaming vs. synchronous vs. async response patterns
- Event-driven AI: queues, webhooks, and async processing
- Vendor abstraction layers to avoid lock-in
