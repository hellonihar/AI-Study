# RAG Systems

Retrieval-Augmented Generation pipelines for grounding LLM outputs in external knowledge.

## Key Topics
- Document ingestion: parsing, chunking strategies, embedding batch jobs
- Retrieval: hybrid search, re-ranking, query rewriting, multi-hop retrieval
- Generation: context window management, citation constraints, faithfulness
- Evaluation: recall@k, faithfulness score, answer relevance, completeness
- Production patterns: caching, fallback chains, circuit breakers
- Advanced RAG: hierarchical retrieval, agentic RAG, self-RAG
