# AI Governance

Policies, processes, and controls for responsible, compliant, and auditable AI deployment.

## Key Topics
- Model risk tiers: classification based on impact, transparency requirements, approval workflows
- Regulatory compliance: EU AI Act, Executive Order, GDPR, HIPAA, SOC 2
- Transparency: model cards, system cards, capability/disability documentation
- Data governance: consent management, retention policies, right-to-deletion
- Ethics: bias auditing, fairness metrics, stakeholder impact assessments
- Audit trails: input/output logging, model version history, decision records
- Organizational structure: AI review boards, centers of excellence, escalation paths
