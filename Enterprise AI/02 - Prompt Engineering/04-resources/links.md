# Resources

## Foundational Papers

| Paper | Year | Contribution |
|---|---|---|
| [Chain-of-Thought Prompting Elicits Reasoning](https://arxiv.org/abs/2201.11903) | 2022 | CoT prompting |
| [Tree of Thoughts: Deliberate Problem Solving](https://arxiv.org/abs/2305.10601) | 2023 | ToT reasoning |
| [ReAct: Synergizing Reasoning and Acting](https://arxiv.org/abs/2210.03629) | 2022 | Reasoning + tool use |
| [Self-Consistency Improves Chain of Thought](https://arxiv.org/abs/2203.11171) | 2022 | CoT with majority voting |
| [Language Models are Few-Shot Learners (GPT-3)](https://arxiv.org/abs/2005.14165) | 2020 | In-context learning |
| [Automatic Prompt Engineer (APE)](https://arxiv.org/abs/2211.01910) | 2022 | Prompt optimization |
| [DSPy: Compiling Declarative Language Model Calls](https://arxiv.org/abs/2310.03714) | 2023 | Programmatic prompt optimization |
| [Jailbroken: How Does LLM Safety Training Fail?](https://arxiv.org/abs/2307.02483) | 2023 | Prompt injection taxonomy |
| [Constitutional AI](https://arxiv.org/abs/2212.08073) | 2022 | Model-level safety |

## Tools & Libraries

| Tool | Description |
|---|---|
| [DSPy](https://github.com/stanfordnlp/dspy) | Automated prompt optimization framework |
| [Outlines](https://github.com/outlines-dev/outlines) | Structured text generation (JSON, regex, grammar) |
| [Guidance](https://github.com/guidance-ai/guidance) | Controlled generation with templates |
| [lm-format-enforcer](https://github.com/noamgat/lm-format-enforcer) | Logit-level schema enforcement |
| [PromptLint](https://github.com/yesdocs/prompt-lint) | Linter for prompt quality |
| [LangSmith](https://smith.langchain.com) | Prompt evaluation and tracing |
| [Weights & Biases Prompts](https://wandb.ai/site/prompts) | Prompt versioning and evaluation |
| [Guardrails AI](https://github.com/guardrails-ai/guardrails) | Input/output guardrails |
