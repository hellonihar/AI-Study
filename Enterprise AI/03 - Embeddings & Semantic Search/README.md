# Embeddings & Semantic Search

Dense vector representations of text and their use in similarity search.

## Key Topics
- Embedding models (sentence-transformers, BGE, E5, OpenAI, Cohere)
- Similarity metrics (cosine, dot product, Euclidean, MIP)
- Training embeddings: contrastive learning, hard negatives, in-batch negatives
- Evaluation: recall@k, MRR, NDCG, hit rate
- Cross-encoders vs. bi-encoders for re-ranking
- Sparse embeddings (SPLADE, BM25) and hybrid search
