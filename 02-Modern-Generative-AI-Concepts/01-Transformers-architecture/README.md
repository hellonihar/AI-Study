# Transformers Architecture

The foundational architecture behind modern large language models and generative AI.

## Key Topics
- Attention is All You Need — original Transformer paper
- Scaled dot-product attention
- Multi-head attention mechanism
- Encoder-decoder architecture
- Positional encoding (sinusoidal, learned, RoPE, ALiBi)
- Feed-forward networks and GLU variants
- Layer normalization and residual connections
- Encoder-only models (BERT, RoBERTa)
- Decoder-only models (GPT, LLaMA, Mistral)
- Encoder-decoder models (T5, BART)
- Efficient attention (Flash Attention, sparse attention)
- Scaling Transformers (model parallelism, tensor parallelism)
